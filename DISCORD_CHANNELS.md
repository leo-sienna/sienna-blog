## 🚀 快速使用

### 对话方式
在任意频道 **@Sienna** 即可与我对话！

```
@Sienna 帮我总结今天的会议要点
@Sienna 这个任务需要多久完成？
@Sienna 分析这张图片
```

### 设置提醒
```
@Sienna 提醒我 明天上午10点 开会
@Sienna 提醒我 每周六 健身
```

### 任务管理
```
@Sienna 添加任务：完成项目报告
@Sienna 列出所有任务
```
